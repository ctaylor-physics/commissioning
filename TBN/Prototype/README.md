TBN Scripts for the Prototype DP
================================
These scripts are intended to be use with data taken with the DP prototype
system that is currently located at the north arm site.   All scripts 
mentioned here uses the lwana-ssmif.txt file for mappings. 

checkTimetags.py
----------------
Check for flow of timetags in a TBN file.

tbnTimeseries.py
----------------
Create plots of I/Q and instantaneous power for a TBN file.

tbnSpectra.py
-------------
Compute integrated spectra for a TBN file.

correlateTBN.py
---------------
Correlate the data in a TBN file and write the output to FITS IDI.
